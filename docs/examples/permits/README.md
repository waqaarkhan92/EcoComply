# Example Permits

This folder contains example environmental permit PDFs used for testing and development.

## Files

- `Application_Bespoke_-_Permit_-_02112022.pdf` - Bespoke permit application example
- `Application_Bespoke_-_Permit_-_02112022 (1).pdf` - Duplicate/alternative version
- `EPR_GP3334CX - Environmental Permit - Original.pdf` - EA Environmental Permit (EPR reference)
- `Permit_-_ZP3537AT.pdf` - Environmental Permit example
- `Permit_London_14_Data_Centre.pdf` - Data centre permit example
- `Permit.pdf` - Generic permit example

## Usage

These PDFs are used for:
- Testing PDF extraction functionality
- Validating obligation extraction accuracy
- Testing OCR capabilities on scanned documents
- Development and QA testing

## Note

These are example documents only. Do not commit sensitive or real customer data to this repository.
